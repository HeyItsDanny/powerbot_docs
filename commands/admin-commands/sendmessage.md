# /sendmessage

**`/sendmessage user`**\
**Beschreibung:** Sendet per Bot eine Nachricht an den User\
**Optionen:** User / Text\
**Permission:** Administrator

**`/sendmessage role`**\
**Beschreibung:** Sendet per Bot eine Nachricht an alle User der Rolle (max. 100)\
**Optionen:** Role / Text\
**Permission:** Administrator
